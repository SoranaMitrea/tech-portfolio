In diesem Projekt habe ich einen Virtual Friction Sensor implementiert, der Sensorfusionsdaten aus einem Arduino-basierten Stack, OBD-II-Daten und Smartphone-GPS synchronisiert. Die Implementierung umfasst die Echtzeit-Sensorfusion, die Berechnung des Reibungskoeffizienten und die Datenpipeline zur Validierung der Ergebnisse. Das README leitet durch die Systemarchitektur, die Arduino-Skizzen und die finale PowerPoint-Dokumentation.
Die Implementierung umfasst zudem eine ML-Pipeline, die auf den fusionierten Sensordaten basiert. Ich habe sowohl Klassifikations- als auch Regressionsmodelle entwickelt, darunter Entscheidungsbäume und neuronale Netze. Die Ergebnisse zeigen, wie die Modelle die Reibungskoeffizienten unter verschiedenen Fahrmanövern prädizieren.
